# 🍽️ Spice Haven – Restaurant Website

## 📌 Project Overview

**Spice Haven** is a modern and responsive restaurant website built using **HTML, CSS, and JavaScript**.
The website showcases an Indian restaurant’s menu, gallery, customer reviews, and contact information in an elegant and interactive design.

This project demonstrates **frontend web development skills**, including responsive layout, smooth scrolling navigation, and attractive UI design.

---

## ✨ Features

### 🏠 Home Section

* Beautiful hero section with background image
* Restaurant introduction and tagline
* "Explore Menu" button for quick navigation

### 🍛 Menu Section

Displays the restaurant's signature dishes with:

* Dish images
* Dish name
* Price
* Description

Example dishes:

* Butter Chicken
* Palak Paneer
* Lamb Rogan Josh
* Tandoori Paneer Pizza
* Chicken Biryani
* Ice Cream

---

### ⭐ Customer Reviews

* Testimonials from customers
* Star ratings
* Customer names and locations

---

### 🖼️ Gallery

* Image gallery showing restaurant interiors and ambience
* Hover zoom effect for better visual experience

---

### 📖 About Section

* Short story about the restaurant
* Describes the restaurant's tradition and passion for authentic Indian cuisine

---

### 📍 Contact Section

Displays restaurant information such as:

* Address
* Opening hours
* Phone number
* Email address

---

### 🎨 Modern UI Design

* Smooth scrolling navigation
* Responsive layout
* Stylish typography using **Google Fonts**
* Hover animations and transitions
* Sticky navigation bar

---

## 🛠 Technologies Used

* **HTML5** – Website structure
* **CSS3** – Styling and layout design
* **JavaScript** – Interactive features
* **Google Fonts** – Typography
* **Unsplash / Online Images** – Food and restaurant images

## ▶️ How to Run the Project

1. Download or clone the project.
2. Open the project folder.
3. Double click **index.html**.
4. The website will open in your browser.

No installation or server setup required.
